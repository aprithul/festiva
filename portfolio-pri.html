2D Wheel Controller

A 2D wheel controller system that is done completely with rigidbodys. Instead of considering a single rigidbody as the car body, I simulated each wheel with a separate rigidbody. This gave the steering a much more believable feel and look. Unfortunately, I lost the source code. The GIF remains from a old facebook post.

Tetris-like Just another tetris game

I hadn't made a tetris game before this. My primary target was to separate the rendering and logic portions as much as possible. That made it very easy to replace the initial ascii character graphics with 3D ones.
