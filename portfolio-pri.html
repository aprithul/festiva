PrEngine2D 2D game engine

PrEngine2D is a 2D C++ game engine that I work on in my spare time as a hobby. It uses SDL for windowing and events, OpenGL 4.1 for rendering and IMGUI for UI. My primary goal with it is to learn engine programming. So don't expect it to power any commercial game soon.

Clone Keen NX port

I ported Clone-Keen (an opensource remake of Commander Keen) to my switch with devkitpro. Clone-Keen was written with the old software renderer based SDL. However, devkitpro only supports SDL2. So the majority of the work went into upgrading it to the newer version of SDL.
